# @ignore

**Tag Kind:** [Modifier](../tags.md#modifier-tags)

Reflections marked with the `@hidden` tag will be removed from the documentation.
It is equivalent to the `@ignore` JSDoc tag.

## Example

```ts
export class Visibility {
    /** @ignore */
    newBehavior(): void;
}
```
